# iOS 风「设置页」完整样式规范（给 Android 用）

> 目标：**一眼像 iOS 设置**
> 不是 Material，不是国产安卓风

---

## 一、整体页面结构

```
Navigation Bar
 ├─ 标题（居中）
 └─ 右侧按钮（完成）

页面背景（浅灰）
 ├─ 分组标题（小字）
 ├─ 圆角卡片
 │    ├─ 行 Item
 │    ├─ 分割线
 │    └─ 行 Item
 └─ 分组标题
```

---

## 二、尺寸规范（非常关键）

### 1️⃣ 页面边距

| 项            | 数值       |
| ------------ | -------- |
| 页面左右 padding | **16dp** |
| 卡片间距         | **12dp** |
| 分组标题与卡片      | **6dp**  |

---

### 2️⃣ 卡片（圆角小盒子）

| 项   | 数值           |
| --- | ------------ |
| 圆角  | **12dp**     |
| 背景色 | #FFFFFF      |
| 内边距 | 无（由 Item 控制） |

> ⚠️ **不要阴影**（iOS 设置页是平的）

---

### 3️⃣ 列表 Item（一行一项）

| 项       | 数值               |
| ------- | ---------------- |
| Item 高度 | **56dp**（最像 iOS） |
| 左右内边距   | **16dp**         |
| 主文字字号   | **17sp**         |
| 副文字字号   | **13sp**         |
| 行间距     | 系统默认             |

#### 布局结构

```
| 16dp | 文本 | （可选副文本） | 控件 | 16dp |
```

---

### 4️⃣ 分割线（非常 iOS 味）

| 项   | 数值        |
| --- | --------- |
| 高度  | **0.5dp** |
| 左缩进 | **16dp**  |
| 右缩进 | **0dp**   |
| 颜色  | #E5E5EA   |

👉 最后一行 **不显示分割线**

---

## 五、控件细节（非常重要）

### 1️⃣ Switch（iOS 风）

| 状态  | 颜色      |
| --- | ------- |
| ON  | #34C759 |
| OFF | #E5E5EA |
| 圆点  | #FFFFFF |

尺寸建议：

```
宽 51dp
高 31dp
```

---

### 2️⃣ 右侧箭头

* 细线箭头
* 垂直居中
* 与文字间距 **8dp**

---

### 3️⃣ 点击态（高亮）

```
背景色：#EDEDED
```

⚠️ **不要波纹（Ripple）**
