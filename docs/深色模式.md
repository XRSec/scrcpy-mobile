# Android 深色模式【直接抄作业】

（Material / 非 Material 都能用）

---

## 一、整体结构（你描述的那种）

```
页面底色（最外层）
 ├─ 横向盒子 / 卡片
 │    ├─ 主文字
 │    ├─ 副文字
 │    └─ Switch / 箭头
 └─ 盒子之间的间隔
```

---

## 二、颜色直接给你（深色模式）

### 1️⃣ 软件**整体底色**

❌ 不要纯黑

```
#000000 ❌
```

✅ 推荐（Android 深色最佳实践）：

```
#121212   ← 最通用
#0F0F0F
```

👉 **最稳**

```
#121212
```

---

### 2️⃣ 横向“盒子 / 卡片”底色

```
#1E1E1E
```

（比背景亮一点，有层级）

可选：

```
#1C1C1E（偏 iOS 风）
#202124（Material 风）
```

---

### 3️⃣ 盒子之间的区域（外容器）

```
同页面底色
#121212
```

👉 不要再加一层灰

---

## 三、文字颜色（非常重要）

### 1️⃣ 主文字（标题 / 设置项）

❌ 不要纯白

```
#FFFFFF ❌
```

✅ 推荐：

```
#EDEDED
```

或：

```
rgba(255,255,255,0.87)
```

---

### 2️⃣ 副文字 / 说明

```
#B3B3B3
```

或：

```
rgba(255,255,255,0.60)
```

---

### 3️⃣ 禁用 / 次要信息

```
rgba(255,255,255,0.38)
```

---

## 四、分割线（能不用就不用）

如果一定要：

```
#2C2C2C
```

高度：

```
0.5dp / 1dp
```

---

## 五、Switch / 滑块颜色（你重点要的）

### 1️⃣ Android 原生 Switch（Material）

#### 开启状态（ON）

```
#4CAF50   （稳）
#34C759   （iOS 风）
```

#### 关闭状态（OFF）

```
系统默认即可
```

---

### 2️⃣ Switch 轨道 & 圆点（推荐值）

| 状态  | 轨道            | 圆点      |
| --- | ------------- | ------- |
| ON  | #4CAF50 (40%) | #4CAF50 |
| OFF | #5A5A5A       | #BDBDBD |

---

## 六、箭头 / 图标颜色

```
#8A8A8A
```

或：

```
rgba(255,255,255,0.60)
```

👉 图标用 **白色模板图**

---

## 七、按钮颜色

### 主按钮

```
#1E88E5
```

### 次按钮

```
#3A3A3A
```

### 不可点击

```
#5A5A5A
```

---

---

## 九、一句话总结（Android 深色铁律）

1️⃣ **背景深，卡片浅**
2️⃣ **文字用白色透明度，不用纯白**
3️⃣ **少线条，多留白（黑）**
