## 多功能按键

- 点击：打开/关闭菜单（就是现有的三个点按钮功能）
- 拖动：移动按钮位置（现有功能保留）
- 长按：桌面（Home）
- 长按+左滑：返回（Back）
- 长按+右滑：后台任务（Recent）
- 长按+上滑：桌面（HOME）
- 长按+下滑：通知栏（Notification）
- 长按：预留功能
- 震动反馈：手势触发时提供触觉反馈
